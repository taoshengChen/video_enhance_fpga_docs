# video_enhance_docs
本项目仓库用于集中管理图像增强项目相关的技术文档。技术文档包括**系列文档**、**设计规范**和**用户手册**。
* **系列文档**：
  * 所有图像增强相关算法的原理和硬件实现
  * 给出通用的图像增强的顶层架构、硬件层次、集成指导
  * 规定图像增强系列必须遵守的通讯协议
  * 给出图像增强系列的硬件资源评估结果
  * 给出图像增强系列的通用寄存器手册
* **设计规范**：
  * 针对特定的应用场景，选用系列文档提到的图像增强算法
  * 针对特定的应用场景，给出图像增强的顶层架构、硬件层次、集成指导
  * 规定在特定场景下，必须遵守的通讯协议
  * 给出图像增强系列的硬件资源评估结果。项目后期，将评估结果替换为硬件统计结果，即实际的硬件资源，并给出资源占比截图
  * 针对选用的图像增强算法，给出寄存器手册
* **用户手册**：
  * 针对特定的应用场景，在用户角度上给出图像增强算法的基本原理、配置流程
  * 规定在特定场景下，在用户角度上给出通讯协议
  * 针对选用的图像增强算法，在用户角度上给出寄存器手册
